Para correr localmente, descargar db con `docker-compose up -d`

- El -d, significa **detached**

* mongoDB URL local:

`mongodb://localhost:27017/teslodb`

## Configurar las variables de entorno

Renombrar el achivo **.env.template** a **.env**
